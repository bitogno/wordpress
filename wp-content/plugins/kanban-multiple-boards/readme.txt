== Description ==

Run multiple boards on the same site.

* Use different boards for different teams, projects, workflows - all from a single site.
* Separate settings for each board allow total customization.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/multiple-boards` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the ‘Plugins’ screen in WordPress.
1. Once the plugin is installed and activated, go to Kanban > Boards.
1. Add as many boards as you need.
1. Once boards are added, click “settings” next to each one, or go to Kanban > Settings and select the board you want to update.
1. Visit your board, and you’ll see your boards in tabs across the top.
